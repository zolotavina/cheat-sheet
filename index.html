HTML Cheat Sheet

Basic Tags

Tag Description Example
<h1> to <h6> Header tags for headings <h1>This is a Heading</h1>
<p> Paragraph tag <p>This is a paragraph.</p>
<a> Anchor tag for links <a href="https://www.example.com">Visit Example</a>
<img> Image tag <img src="image.jpg" alt="Description">
<ul> <li> Unordered list and list item <ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>
<ol> <li> Ordered list and list item <ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

Form Tags

Tag Description Example
<form> Form container <form action="/submit" method="post">...</form>
<input> Input field <input type="text" name="username" placeholder="Enter your username">
<textarea> Textarea for multiline input <textarea rows="4" cols="50">Enter text here...</textarea>
<select> <option> Dropdown list <select>
  <option value="option1">Option 1</option>
  <option value="option2">Option 2</option>
</select>
<button> Button element <button type="submit">Submit</button>
<label> Label for form elements <label for="username">Username:</label>
